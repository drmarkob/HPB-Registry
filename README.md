# HPB Patient Registry

A secure, structured registry for managing patient data, specifically designed for HPB (Hepatobiliary-Pancreatic) clinical records.

## 🩺 Overview
This application is built to streamline the collection and analysis of patient histories, treatments, and follow-ups. It is currently developed and tested on Linux Mint.

## 🛠️ Key Features
* **Patient Records Management:** Create and update detailed patient profiles.
* **Clinical Data Indexing:** Efficiently track specific HPB medical parameters.
* **Search Functionality:** Filter patients by diagnosis or treatment dates.
* **Privacy Focused:** Designed to keep sensitive data local.
