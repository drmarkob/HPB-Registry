📖 How to Import into SPSS

  1. Click "Export Complete Dataset" above
  2. Save the Excel file to your computer
  3. Open SPSS
  4. Go to File → Open → Data
  5. Change file type to "Excel (*.xlsx)"
  6. Select your downloaded file
  7. Choose the sheet you want to analyze (e.g., "Patients")
  8. Click OK - SPSS will automatically detect variable types
💡 Tip: For survival analysis, use the "Follow_Up" sheet with survival_months and alive variables.
